Links

Links are used to connect to another webpage and use a tags with href and targetattributes. For example, Golden State Student Portal. target="_blank" allows the webpage to open in a new tab.

Links can also be used to nagivate to a section of the webpage using href="#id" in the a tags where top is an id attribute in the opening tag of the page element want to navigate to. For example, click here to go to the top of the page.

To link HTML to CSS page, use link self-closing element with the following attributes in the head of the HTML document:

  • href="style.css"
  • type="text/css"
  • rel="stylesheet"

Lists

Lists can be ordered (od) or unordered (ud). To change bullets in the ordered list, use type attribute with the following values:

  1. type="1"
  2. type="A"
  3. type="a"
  4. type="I"
  5. type="i"

To change bullets in the unordered list, use style attribute with the following CSS values:

  • style="list-style-type: disc;"
  • style="list-style-type: square;"
  • style="list-style-type: circle;"
  • style="list-style-type: none;"

Font Awesome

Font Awesome is used to add icons to a webpage. There are two methods for adding Font Awesome by linking into HTML page in the head section:

  • Link to external CDN, Bootstrap CDN
  • Link downloaded local Font Awesome CSS file

Linking is done the same way as any other CSS file using link element
[link type="stylesheet" type="text/css" href="link to local file or Bootstrap CDN"]

Font Awesome icons are added to HTML page using [i] tag and fa-icon-name and fa classes
[i class="fa fa-save"][/i] where fa-save is a type of icon. Use Font Awesome Cheatsheet for a list of icons and names.

Size of icons can be increased with additional classes:

  • fa-lg: 33% increase
  • fa-2x: 2x increase
  • fa-3x: 3x increase
  • fa-4x: 4x increase
  • fa-5x: 5x increase
[i class="fa fa-facebook fa-2x"

Favicons

Favicon is a version of the logo made from a logo using a favicon maker (e.g. favicon-generator.org). Image is converted to the favicon and then pasted in the root website directory with the following code added to the head section of html page.

[link rel="icon" href="./.ico" type="image/x-icon"]
